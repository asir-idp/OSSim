Exemple de planificació multiprogramació amb E/S

Processos de diferents durades a la cua de preparats, els processos combinen ràfegues de CPU i E/S. Durant les ràfegues de E/S el següent procés s'executa.


ProcésRàfegues
(durada i tipus)
P12 cpu + 2 io + 1 cpu
P24 cpu
P33 cpu + 2 io + 2 cpu
P43 cpu

Arriben en l'ordre de la taula, tots a temps 0.

Observacions:

  • Els processos comencen i acaben sempre amb ràfegues de cpu
  • Els processos que abandonen la cpu per realitzar operacions d'E/S, un cop finalitzades tornen al final de la cua
