Monoprogramming scheduling example with I/O

Different duration processes in the ready queue, processes mix CPU and I/O bursts. Processes run until complete, during I/O bursts does not enter processor, it is idle.


Process Bursts
(Duration and type)
P1 2 CPU + 2 I/O + 1 cpu
P2 4 cpu
P3 3 cpu + 2 I/O + 2 cpu
P4 3 cpu

All processes arrive in the table order at time 0.

Comments:

  • The process always begins and ends with cpu bursts
  • While a process leave the CPU to perform I/O, the processor is idle
