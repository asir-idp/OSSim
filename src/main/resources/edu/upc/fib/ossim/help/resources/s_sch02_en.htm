Non preemptive SJF scheduling example

Different duration processes in the ready queue, processes have only CPU bursts. A short process arrives later.

All processes arrive in the table order, process "P5" arrives at time 4.

Comments:

  • Process are sorted by its duration, first those with a shorter duration
  • At the time it arrives "P5" (duration 2 units), is running "P2" (3 units left)
  • Algorithm is not preemptive, then "P2" keeps running and "P5" is queued
