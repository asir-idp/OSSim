Exemple de planificació monoprogramació amb E/S

Processos de diferents durades a la cua de preparats, els processos combinen ràfegues de CPU i E/S. Els processos s'executen completament, durant les ràfegues de E/S el processador queda ociós.


ProcésRàfegues
(durada i tipus)
P12 cpu + 2 io + 1 cpu
P24 cpu
P33 cpu + 2 io + 2 cpu
P43 cpu

Arriben en l'ordre de la taula, tots a temps 0.

Observacions:

  • Els processos comencen i acaben sempre amb ràfegues de cpu
  • Mentre els processos abandonen la cpu per realitzar operacions d'E/S, el processador queda ociós
