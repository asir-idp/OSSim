Example: Segmentation, partial allocation

The processes are divided into segments, each segment can be a different size. The segments are loaded into memory independently, a partition of the exact segment size is created. The division is functional and is based on the logical process structure: data, stack, code, etc ...

Not necessarily all the segments have to be loaded into memory for the program to work, while a segment is not used it can be swapped out.
